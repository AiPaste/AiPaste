Why it exists

Clipboard tools should feel invisible until the exact second you need them.

AiPaste is tuned for frequent copy-paste work: engineering notes, shell commands, prompts, reusable snippets, and the half-finished fragments you do not want to lose while context-switching.

01

Fast history with source awareness

AiPaste watches the system pasteboard, keeps recent text clips locally, and tracks the frontmost app so your history is easier to scan.

02

Pin what matters

Save the commands, links, and answers you reuse every day without re-copying them again and again.

03

Group by project

Move items into named groups, color them, rename them, and keep work clips separate from everything else.

04

Paste back into the active app

Use Accessibility-backed paste-to-app actions when you need speed more than manual switching.

05

Ignore noisy apps

Define ignore rules for apps or bundle IDs so transient or sensitive sources do not pollute history.

06

Native distribution path

Install with Homebrew, download packaged releases, and get automatic update checks from GitHub Releases.
