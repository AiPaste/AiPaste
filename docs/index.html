AUTOMATE

Use the app when you want UI. Use the CLI when you do not.

Show or hide the panel, paste a selected clip, manage groups, and change configuration from terminal commands that match the same workflow as the desktop app.

$ ./bin/aipaste panel toggle $ ./bin/aipaste items paste 1 $ ./bin/aipaste groups rename group-1 archive $ ./bin/aipaste config set run-in-background false

FAQ

Built for native Mac workflows.

Does it store data locally?

Yes. AiPaste is designed as a local-first clipboard tool for macOS.

Can I automate it?

Yes. The bundled CLI covers panel actions, item actions, groups, ignore rules, and configuration.

Can it paste back into another app?

Yes, with macOS Accessibility permission enabled for paste-to-app actions.

How do I preview this site locally?

Run ./scripts/serve_website.sh and open the local address it prints.
